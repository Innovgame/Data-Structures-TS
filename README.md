# Data Structures

## init Project

### install webpack and webpack-cli

1.webpack

```sh
# 默认安装最新版本
$ npm install --save-dev webpack
# 安装指定版本
$ npm install --save-dev webpack@<version>
```

2.webpack-cli (4+版本需要安装)

```sh
# 4+版本需要独立安装webpack-cli
$ npm install --save-dev webpack-cli
```
